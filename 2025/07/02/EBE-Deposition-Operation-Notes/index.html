本文为电子束蒸发（EBE）沉积操作的标准实验流程笔记，支持中英文切换阅读。
This post provides a bilingual operation manual for EBE thin-film deposition used in cleanroom microfabrication.

EBE（电子束蒸发）沉积流程包括以下主要步骤，建议全流程约需 30-40 分钟，根据材料数量及沉积层数可能有所变化。

1. 到场检查

  • 检查主腔室真空度（优于 1E-7 Torr）
  • 检查Load Lock 阀门是否关闭（XFER VALVE 按钮应为红色）
  • 检查Load Lock 真空度（一般应优于 1E-5 Torr）

2. LoadLock 装样

  1. 关闭 Load Lock 推杆开关，等待真空回到常压（约 7.5E2 Torr），打开盖子；
  2. 取出 hold ring，到操作台贴装样品至 dummy wafer；
  3. 将 dummy wafer 安装至 hold ring，氮气吹洗；
  4. 整体装入 Load Lock holder；
  5. 轻压 lid，打开推杆开关抽真空，直至低于 1E-6 Torr。

2-a. 参数设置

  • E-beam Sweep Controller：根据材料选择对应 beam，设置 Program Number 和 Pocket Number，确认 Pattern 正确；
  • Deposition Controller：在Process Menu中旋钮选择材料，退出到主界面，进入 Quick Edit，设置速率和目标厚度，确认并返回主界面。

3. 送样进入主腔室

  • Motor Control 设为 Local，转速设为 0；
  • 摇杆调整 holder claws 角度与高度（对齐 F line 和 Red Line）；
  • 打开 XFER VALVE，插入 transfer arm 至顶；
  • 提升 claws 超过 Blue Line，拉出 holder，关闭阀门；
  • 提升 claws 至顶部，设转速 50 rpm，切换至 Remote 模式，关闭挡板。

⚠️ 操作全程需目视避免碰撞，如有阻碍，及时退回调整 claws。

4. 执行沉积

  • 检查 Motor Control 为 Remote Mode；
  • 再次确认 beam 设置与 deposition 参数；
  • 点击 Run Process，实时监测沉积速率、厚度、功率、电流、电压；
  • 程序自动停止。多层材料请重复参数设置与运行步骤。

4-a. 日志记录

  • 记录沉积速率与 emission current（从 beam 控制器读取），填写至 log book。

5. 取样

  • 停转速至 0，切回 Local Mode；
  • 打开挡板，降低 claws（不低于 Blue Line），调节角度对齐 F Line；
  • 确保 Load Lock 真空低于 1E-6 Torr，打开 XFER VALVE；
  • 插入 transfer arm，降低 claws 至 Red Line，完成交接；
  • 拉出 arm，关闭阀门；
  • 关闭推杆开关放气，待气压至 7.5E2 Torr；
  • 打开盖子，取出 wafer，卸样，归还 holder ring 并重新抽真空。

6. 收尾步骤

  • 操作台卸样，dummy wafer 归位，样品收纳；
  • 待 Load Lock 真空优于 1E-4 Torr；
  • 在 log book 记录离开时间，完成操作。

The EBE (Electron Beam Evaporation) process consists of the following sequential steps. The total process typically takes about 30–40 minutes depending on the number of materials and thickness layers.

1. Initial Check 【1 min】

  • Check the main chamber vacuum (< 1E-7 Torr)
  • Check if the Load Lock valve is closed (XFER VALVE button should be RED)
  • Check the Load Lock vacuum level (typically < 1E-5 Torr)

2. Load Lock Sample Loading 【10 min】

  1. Turn off the Load Lock pusher, wait for the pressure to rise to ambient (~7.5E2 Torr), then open the lid.
  2. Remove the hold ring and mount the sample onto the dummy wafer on the workbench.
  3. Fix the dummy wafer onto the hold ring, blow clean with nitrogen.
  4. Place the entire assembly onto the Load Lock holder.
  5. Gently press the lid and turn on the pusher to pump down until pressure < 1E-6 Torr.

2-a. Parameter Setup 【3 min, in parallel during vacuuming】

  • E-beam Sweep Controller: Select the beam and corresponding program and pocket numbers based on material list. Confirm the pattern is correct and ensure the program button is activated.
  • Deposition Controller: Find the "Process Menu", select the material by knob. Back to main meuw, find the "Quick Edit" via Next Menu, use the knob set the rate and final thickness, confirm and return to main interface.

3. Transfer to Main Chamber 【3 min】

  • Set Motor Control to Local Mode and rotation speed to 0.
  • Use joystick to align holder claws angle with F line and height with Red Line.
  • Open XFER VALVE, slide in transfer arm to the end, raise claws above Blue Line and extract the holder.
  • Close the valve, continue lifting claws to top mark, switch to Remote Mode at 50 rpm, then close the shutter.

⚠️ Monitor visually at all times. If collision or obstruction occurs, retract and adjust the claw angle immediately.

4. Deposition Process 【10 min】

  • Confirm Motor Control is in Remote Mode.
  • Ensure E-beam and deposition settings are correct.
  • Click Run Process. Monitor deposition rate, thickness, power, emission current, and voltage in real-time.
  • The process stops automatically upon reaching the target thickness.
  • For multilayer metals, repeat parameter setup and execution steps.

4-a. Log Recording 【1 min, in parallel during deposition】

  • Record deposition rate and emission current from the E-beam controller into the log book.

5. Sample Retrieval 【4 min】

  • Reduce rotation speed to 0 and return to Local Mode.
  • Open shutter, lower claws (not below Blue Line), align angle to F Line.
  • Ensure Load Lock pressure is below 1E-6 Torr, open XFER VALVE.
  • Insert transfer arm, lower claws to Red Line for handover.
  • Slowly retract arm, close XFER VALVE.
  • Turn off Load Lock pusher to vent, wait until ambient pressure (~7.5E2 Torr).
  • Open lid, remove wafer, unmount sample, return holder ring and repump.

6. Final Steps 【2 min】

  • Remove the sample, restore dummy wafer, and store the sample.
  • Wait for Load Lock vacuum to drop below 1E-4 Torr.
  • Record the end time in the log book and complete the procedure.
